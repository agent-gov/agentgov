How It Works

One command. Total visibility.

$ npx @agentgov/cli audit
🔍

Scan

Walks your codebase using fast-glob. Respects .gitignore. Analyzes Python & TypeScript files for agent framework patterns.

🎯

Detect

Multi-signal confidence scoring: imports, instantiation, config files, dependencies, environment variables. Reports agents scoring 0.4+.

⚠

Risk & Report

Flags governance risks: no owner, stale agents, email capability, DB access, agent spawning. Outputs terminal, JSON, or HTML reports.

Every decision is reversible
